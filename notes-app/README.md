# Notes App

A local notes-taking app organized by projects. Runs as a local web server + browser UI.

## Prerequisites

- Node.js (v18+)

## Setup

```bash
cd notes-app
npm install
```

## Run (Development)

```bash
npm run dev
```

This starts:
- Vite dev server at `http://localhost:1420` (frontend)
- Express API server at `http://localhost:3001` (backend)

Open `http://localhost:1420` in your browser.

## First Launch

1. Enter a folder path where your data will be stored (e.g., `G:\MyNotes`)
2. Click "Get Started"
3. Start creating projects!
